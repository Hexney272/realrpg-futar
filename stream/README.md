# Stream mappa - Custom Prop-ok

Ide kell elhelyezni a `prop_shop_locker` custom prop fájljait.

## Szükséges fájlok:
- `prop_shop_locker.ydr` - A 3D modell fájl
- `prop_shop_locker.ytd` - Textúra fájl (ha van)

## Honnan szerezheted be:
- https://forum.cfx.re/t/prop-shop-locker/5148437

## Telepítés:
1. Töltsd le a prop fájlokat
2. Helyezd el ebben a `stream/` mappában
3. A fxmanifest.lua automatikusan stream-eli
